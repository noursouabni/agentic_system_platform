# Agentic Campaign Intelligence Platform

A multi-agent AI platform that transforms a marketing campaign brief into research insights, creative assets, media planning recommendations, and optimization actions.

## Project Goal

This project demonstrates how Agentic AI can support autonomous campaign creation and real-time optimization through a structured workflow:

1. Understand the campaign brief
2. Delegate tasks to specialized agents
3. Generate campaign assets
4. Simulate campaign performance
5. Recommend optimization actions

## Tech Stack

- Python
- Streamlit
- FastAPI
- Pandas
- Plotly
- Pydantic
- LLM integration planned
